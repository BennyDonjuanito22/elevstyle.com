Refund & Return Policy

Effective Date: February 11, 2026

Last Updated: February 11, 2026

Elev Style is a trade name operated by ASI Innovative Solutions LLC.

1. Return Policy

Most items can be returned within 30 days of delivery.

2. Return Eligibility

2.1 Eligible

  • Items in original, unworn condition
  • Items with all tags attached
  • Items in original packaging
  • Items returned within 30 days

2.2 Not Eligible

  • Worn, washed, or altered items
  • Items without tags
  • Intimates, swimwear (for hygiene)
  • Items marked "Final Sale"
  • Gift cards

3. How to Return

  1. Email support@elevstyle.com with order number
  2. Receive return authorization and instructions
  3. Pack items securely
  4. Ship using provided instructions
  5. Refund processed within 5-7 business days of receipt

4. Return Shipping

  • Defective/wrong item: We provide prepaid label
  • Changed mind: Customer pays return shipping

5. Refunds

  • Processed to original payment method
  • 5-7 business days after we receive return
  • May take additional time to appear in account

6. Exchanges

Return for refund and place a new order for different size/color.

7. Damaged Items

Report within 48 hours with photos. We'll arrange replacement or refund.
